# OrangeHRM Automation Assessment

## Overview
This project automates the OrangeHRM demo application using Selenium WebDriver with Java and Page Object Model (POM).

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
